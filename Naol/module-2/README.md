# Naol - Module 2: Address and Ether Interaction

## What this module teaches
In this module, you learn how Solidity contracts send and receive value and how they interact with external addresses and contracts.

## Highlights
- Low-level call behavior and success handling
- Building function selectors and signatures
- Fallback and receive functions for Ether intake
- Escrow flow and approval concepts
- How to protect contract operations with checks
- Recognizing and handling transaction reverts
- Sending Ether safely and cleaning up contracts

## Structure
- `Calling Contracts/`
- `Escrow/`
- `Reverting Transactions/`
- `Sending Ether/`

## How to use it
1. Study the example contracts to understand data flow.
2. Add validation and security checks where required.
3. Run the provided tests and inspect each Ether transfer.

## Note
This module is focused on real contract interaction patterns, not just language syntax.
