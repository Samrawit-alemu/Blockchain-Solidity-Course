# Naol - Module 1: Solidity Fundamentals

## What this module teaches
This module introduces Solidity from a practical perspective. You will implement basic contracts, work with data types, and practice writing simple functions.

## Highlights
- Solidity contract layout and declaration
- Boolean checks and conditional logic
- `uint` and `int` numeric operations
- Strings and enumerated types
- Function arguments, return values, and visibility
- Difference between read-only and pure helper functions
- Using debug output while testing

## Structure
- `Booleans/`
- `Unsigned Integers/`
- `Signed Integers/`
- `String Literals/`
- `Enum Type/`
- `Arguments/`
- `Increment/`
- `View Addition/`
- `Console Log/`
- `Pure Double/`
- `Double Overload/`

## How to use it
1. Open a contract in the lesson folder.
2. Complete missing operations or return values.
3. Run test cases to verify your implementation.

## Note
This module is all about learning the Solidity building blocks that power later modules.
