# Naol Solidity Course

This folder contains Naol's version of the Solidity course. Each module is written with a focus on practical Solidity patterns, step-by-step exercises, and clear explanations.

Modules included:
- `module-1` — Solidity language basics and function practice
- `module-2` — Address handling, contract interaction, and Ether movement
- `module-3` — Reference types, storage, and structure modeling
- `module-4` — Applied Solidity with inheritance and voting mechanics

Each module has its own `README.md` to guide you through the lessons and highlight the most important concepts.
